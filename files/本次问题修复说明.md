# 本次问题修复说明

## 一、后端逻辑修复

### 1. 文件名与目录名校验

原问题：
- 命令层使用固定长度数组接收文件名，存在超长输入导致缓冲区溢出的风险。
- 目录项名称长度为 14 字节，旧逻辑会截断长文件名，可能出现“创建时被截断、后续查找失败或误操作”的问题。

修改内容：
- 在 `name.c` 中新增 `valid_dir_name()`，统一校验目录项名称。
- 目录项名称必须满足：
  - 非空；
  - 长度小于 `DIRSIZ`，即最多 13 个字符；
  - 不包含 `/`。
- `iname()` 在写入目录项前统一拒绝非法名称。
- 命令解析层改为使用较大临时缓冲区，再交由底层校验，避免截断后误操作。

涉及文件：
- `filesys.h`
- `name.c`
- `main.c`

### 2. 目录项数量维护

原问题：
- 删除文件或目录时，只有删除最后一个目录项才减少 `dir.size`。
- 创建文件、目录、硬链接、软链接时无条件 `dir.size++`。
- 多次删除和复用空洞目录项后，`dir.size` 可能虚增，导致目录大小不准确。

修改内容：
- 在 `name.c` 中新增：
  - `dir_entry_occupy()`：占用目录项时只在需要时扩展 `dir.size`。
  - `dir_entry_remove()`：删除目录项后自动回收末尾连续空洞。
- 文件创建、目录创建、删除、硬链接、软链接、跨目录移动均改用统一目录项维护函数。

涉及文件：
- `filesys.h`
- `name.c`
- `creat.c`
- `dir.c`
- `delete.c`
- `link.c`
- `mv.c`

### 3. 命令参数解析

原问题：
- `mkdir a b` 这类命令会把完整参数字符串传入后端，可能创建出包含空格的目录名，后续难以访问。
- 多处 `sscanf("%s", ...)` 没有限制输入长度。

修改内容：
- 增加单参数解析辅助函数 `parse_single_name_arg()`。
- `mkdir`、`cd`、`rmdir` 现在只接受一个参数，多余参数会输出用法提示。
- 命令名读取增加长度限制，避免命令缓冲区溢出。

涉及文件：
- `main.c`

### 4. 当前路径显示保护

原问题：
- `cd` 成功后直接使用 `strcat()` 拼接 `current_path`，深层目录可能导致路径缓冲区溢出。

修改内容：
- 新增路径长度检查函数。
- 在真正切换目录前先判断路径显示是否会超长。
- 路径拼接改为通过安全的格式化逻辑完成。

涉及文件：
- `main.c`

### 5. 软链接递归保护

原问题：
- 软链接打开逻辑递归调用 `aopen()`。
- 如果出现软链接自环或循环引用，会无限递归。

修改内容：
- 将软链接解析改为内部递归函数 `aopen_depth()`。
- 最大软链接解析深度限制为 8。
- 超过限制时输出 `Too many levels of symbolic links` 并返回打开失败。

涉及文件：
- `open.c`

### 6. 前端数据快照接口

原问题：
- 前端刷新一次界面需要连续请求多个内部命令，增加 socket 往返次数。

修改内容：
- 后端新增内部命令 `__snapshot__`。
- 一次返回磁盘块、inode 列表、当前目录、目录树和系统统计。

涉及文件：
- `main.c`
- `frontend/fs_client.py`

## 二、前端优化

### 1. daemon 端口动态分配

原问题：
- 前端固定使用 `127.0.0.1:9090`。
- 如果端口被占用，可能启动失败或误连旧 daemon。

修改内容：
- `FsClient` 默认自动寻找本地空闲端口。
- 启动 daemon 时使用动态端口。

涉及文件：
- `frontend/fs_client.py`

### 2. 刷新性能优化

原问题：
- 刷新界面需要多次 socket 请求。

修改内容：
- 前端刷新改为调用 `__snapshot__`。
- 解析快照中的多个 section 后填充界面。

涉及文件：
- `frontend/fs_client.py`

### 3. 表格交互优化

修改内容：
- inode 表和目录表改为只读。
- 支持整行选择。
- 支持单行选择。
- 支持列排序。
- 开启隔行背景色。

涉及文件：
- `frontend/main.py`
- `frontend/assets/style.qss`

### 4. 磁盘块显示优化

修改内容：
- 增加颜色图例：
  - 系统保留；
  - 已占用；
  - 空闲。
- 鼠标悬浮磁盘块时显示块号和状态。
- 点击磁盘块后继续显示当前选中块信息。

涉及文件：
- `frontend/main.py`
- `frontend/widgets/disk_grid.py`
- `frontend/assets/style.qss`

### 5. 命令终端体验优化

修改内容：
- 命令输入框支持上下键查看历史命令。
- 执行命令期间临时禁用相关按钮，避免重复提交。

涉及文件：
- `frontend/main.py`
